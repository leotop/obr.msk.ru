Школы Москвы альфа-версия

Выбор школы для ребёнка — это важный вопрос. Часто решение по нему принимается нерационально. Ниже приведены некоторые показатели московских школ, которые могут сделать выбор более качественным. На данные момент информация собрана для топ-300 школ по версии Министерства образования. Результаты ЕГЭ всё ещё труднодоступны, поэтому встречаются только у 30% школ.
Данные частично предоставлены Министерством образования, частично взяты из публичный докладов школ и из социальных сетей. Разработкой занимаются Российские школы и Лаборатория анализа данных Кукушкина Александра. Адрес для вопросов и комментариев — qa@obr.msk.ru.
